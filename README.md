# hunch

A Swift command line tool to access the Notion.so API.

## Installation

Install globally using [Mint](https://github.com/yonaskolb/Mint):

```bash
mint install adamwulf/hunch@main --force
```

## References

Some URLSession/request code from https://github.com/maeganwilson/NotionSwift.

Notion API: https://developers.notion.com/reference/block
